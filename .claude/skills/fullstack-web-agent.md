## Full-Stack Web Agent Skill

Launch the Full-Stack Web Agent to transform console apps into multi-user web applications. This agent serves as the authoritative generator for creating full-stack web applications using Next.js 16+ with App Router, FastAPI with RESTful endpoints, SQLModel with Neon PostgreSQL, and Better Auth with JWT.

### When to Use
- When transforming console apps to web apps
- When implementing multi-user functionality
- When building Next.js frontend applications
- When developing FastAPI backend services
- When managing database integration
- When implementing authentication systems

### Capabilities
- Transforming console apps to web applications
- Implementing multi-user functionality
- Building Next.js frontend with App Router
- Developing FastAPI backend with RESTful endpoints
- Managing SQLModel with Neon PostgreSQL
- Implementing Better Auth with JWT

### Options
- `--frontend`: Focus on Next.js frontend development
- `--backend`: Focus on FastAPI backend development
- `--database`: Focus on SQLModel and database integration
- `--auth`: Focus on authentication system
- `--transform`: Focus on console-to-web transformation

### Best Practices
- Follow full-stack integration best practices
- Maintain security with proper authentication
- Optimize performance for both frontend and backend
- Ensure scalability for multiple users
- Maintain clean architecture patterns
- Follow Next.js and FastAPI best practices
